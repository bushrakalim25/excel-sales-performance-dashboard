# 📊 Excel Sales Performance Dashboard

## 📌 Project Overview

This project is an **End-to-End Sales Performance Dashboard** developed in **Microsoft Excel**. The dashboard transforms raw sales data into meaningful business insights using Excel's data analysis and visualization features.

It is designed to help businesses monitor sales performance, compare employee achievements, track target completion, and identify areas that require improvement.

---

## 🚀 Features

* Interactive Excel Dashboard
* Sales Performance Analysis
* Employee-wise Sales Tracking
* Target Achievement Analysis
* Underperformer Identification
* KPI Monitoring
* Pivot Tables
* Pivot Charts
* Slicers for Interactive Filtering
* Clean and Professional Dashboard Design
